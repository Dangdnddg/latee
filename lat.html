BOX MODEL

CSS box model is a container which contains multiple properties including borders, margin, padding and the content itself. It is used to create the design and layout of web pages. It can be used as a toolkit for customizing the layout of different elements. The web browser renders every element as a rectangular box according to the CSS box model. Box-Model has multiple properties in CSS like : borders, margins, padding, Content.

This is an example of BOX MODEL having
width: auto; height: auto; border: 15px ; padding: 50px; margin: 75px.
